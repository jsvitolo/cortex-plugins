Open the Cortex TUI (Terminal User Interface).

## Usage

```bash
cx ui
```

## Features

The TUI provides:
- **Kanban Board**: Visual task management with 4 columns
- **Task Detail**: View and edit task details
- **Memory Browser**: Navigate and view memories
- **Worktrees**: Manage git worktrees for tasks

## Keyboard shortcuts

### Kanban
- `h/l`: Move between columns
- `j/k`: Move between tasks
- `Enter`: Open task detail
- `m`: Move task to another column
- `s`: Start task (move to in_progress)
- `x`: Mark as done
- `d`: Delete task
- `w`: Open worktrees view
- `m`: Open memory browser
- `?`: Toggle help
- `q`: Quit

### Memory Browser
- `j/k`: Navigate memories
- `Enter`: Open memory detail
- `t`: Toggle memory type filter
- `r`: Refresh list
- `Esc`: Go back
